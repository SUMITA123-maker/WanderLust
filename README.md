# 🏕️ Wanderlust - Mini Airbnb for Campgrounds

**Wanderlust** is a full-stack web application inspired by Airbnb, tailored specifically for discovering and sharing campgrounds. Users can browse, add, edit, and review campgrounds with rich media and location data.

> 🚧 **Note**: This is an ongoing project, currently under active development.

---

## 🔑 Key Features

- 🔐 User authentication and authorization (Register/Login/Logout)
- 🏕️ Create, edit, and delete campground listings
- 💬 Post and manage reviews on campgrounds
- 🖼️ Responsive UI with dynamic EJS templates
- 🧩 MVC structure with robust validation and error handling

---

## 🧰 Tech Stack

- **Frontend**: HTML, CSS, Bootstrap, EJS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (via Mongoose)
- **Authentication**: Passport.js
